# gc(git checkout)

The same liek `cd -`, will checkout the last branch.

How to use:

```
# current branch: master
$ gc br1

# now you are on branch br1


# to back to master branch, you can run gc -
$ gc -
```

Attention: if you have many git repos, this will affect all.
